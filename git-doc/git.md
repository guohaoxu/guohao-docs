# git入门笔记

```sh
$ git
```

linux系统：
`sudo apt-get install git`
可以通过源码安装,下载源码，解压，输入：
`./config`,`make`,`sudo make install`

### 设置
```sh
$ git config --global user.name "guohao"
$ git config --global user.email "guohao0817@gmail.com"
```

### 创建版本库
```sh
$ mkdir firstgit
$ cd firstgit
$ git init
```
